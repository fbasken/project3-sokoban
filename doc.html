Documentation

One issue I ran into while developing this game was Javascript's lack of real abstract class support, and furthermore, support for multiple inheritance. I had to repeat the same sections of code, however it was fairly minor. I would definitely choose a different language were I to develop a similar project in the future, probably opting to use TypeScript for its strong typing and support for object-oriented patterns.
